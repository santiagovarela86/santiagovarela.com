This collection of repositories is a practical environment for experimenting with a broad DevOps toolchain. It covers source control and continuous integration alongside containerized workloads, Kubernetes deployments, and infrastructure defined with Terraform.

The AWS examples extend into managed services such as S3, Lambda, SQS, DynamoDB, and API Gateway, while the Azure work includes AKS. Together, the projects provide a place to test how application delivery, infrastructure automation, and cloud services fit into a repeatable deployment workflow.
